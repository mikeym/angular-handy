This sample project includes things I often use to help me get started quickly on prototypes. Good stuff includes:

  • Angular JS 1.5 web framework
  • Bootstrap 3 responsive web styles
  • Moment.js 2.14 clever time and date functions
  • Font Awesome 4.6 cute little icons
  • Jasmine 2.4 test framework
  • Karma 1.0 unit test runner

My goals for this project were to put together a small starter app that:

  • made a call to an external API
  • rendered a collection into a table
  • included simple date and time functions via moment.js
  • included responsive styling via Bootstrap
  • included cute icons via Font Awesome
  • had well-structured code
  • had good unit tests

See the angular-handy project on GitHub for more information.
